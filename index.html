Na Unidade Curricular de Programação Web abordaremos os princípios básicos associados ao desenvolvimento para a web, tanto do lado back-end como front-end

Apreender-se-ão as seguintes tecnologias 🌐:

  • CSS
  • Python
  • Django
  • JavaScript

De forma a consolidar todos os conhecimentos, será feito um laboratório semanal, de entrega obrigstória, de aplicação da matéria leccionada. Irá ser desenvolvido também um projeto.
